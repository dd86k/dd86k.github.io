ddtty.js - v0.2.0-dev

Please note that ddtty.js is a prototype that is nowhere near ready for deployment. It is will under development.

Still needs:

  • Display buffer
  • ReadLine()
  • ReadKey()
  • Write() (with buffer)
  • SetCursorPosition()
  • Resizable buffer

Create

First step: Create the DDTTY object.

Best practice is to bind it to window in mid-script. Like this: window['Example'] = DDTTY("cont");

window.Example also works. "cont" is the parent's id, the container for Example.

Then, you can do Example.Write("Hi");, specifying window is optional and/or not needed at all.

If no parent is specified, the HTML will be added to body.

Initialize

Initializing the object is easy: Example.Initialize();

Note that it is very important to create the object first.

Text

JS Exemple: Example.Write(object); or Example.WriteLine(object);

In this exemple: Write anything!

Color

JS Exemple: (Any CSS color scheme) Example.ForegroundColor = "white"; Example.BackgroundColor = "rgb(23, 54, 244)";

In this exemple: Color names, #FFFFFF, rgb(), and hsl() are all accepted.
